## 🚀 How to Run

1. **Install Anaconda (Recommended)**
   Download and install **Anaconda Navigator** from the official website. It helps manage environments and dependencies easily.

2. **Clone the repository:**

   ```bash
   git clone https://github.com/CodeVoot/Agentic-Systems.git
   ```

3. **Navigate to the project folder:**

   ```bash
   cd <repo-name>
   ```

4. **Create a virtual environment using Conda:**  (Optional)

   ```bash
   conda create --name environment_name
   ```

5. **Activate the environment:** (Optional)

   ```bash
   conda activate environment_name
   ```

6. **After successfully activate the virtual environment or installing the Anaconda in the system, just run the command to run the jupyter notebook and open the root folder:**

   ```bash
    jupyter notebook
   ```
